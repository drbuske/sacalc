Section 5.6 Trigonometric Integrals

Motivating Questions

  • How do we solve integration problems involving products and powers of \(\sin x\) and \(\cos x\text{?}\)

  • How do we solve integration problems involving products and powers of \(\tan x\) and \(\sec x\text{?}\)

In this section we look at how to integrate a variety of products of trigonometric functions. These integrals are called trigonometric integrals. They are an important part of the integration technique called trigonometric substitution. This technique allows us to convert algebraic expressions that we may not be able to integrate into expressions involving trigonometric functions, which we may be able to integrate using the techniques described in this section. In addition, these types of integrals appear frequently when we study polar, cylindrical, and spherical coordinate systems.

Subsection 5.6.1 Integrating Products and Powers of \(\sin x\) and \(\cos x\)

One type of integral commonly appearing in practice is that of the form

\begin{equation*} \int \cos^j x \sin^k x \ dx \end{equation*}

where \(j\) and \(k\) are nonnegative integers. Preview Activity 5.6.1 demonstrated how to handle the case in which \(k\) is odd - we break off one power of \(\sin x\) and use the trigonometric identity \(\sin^2x + \cos^2x = 1\) to convert the rest of the integrand so that it involves powers of \(\cos x\) followed by use of the substitution \(u=\cos x\text{.}\)

A similar strategy is useful when the power of \(\cos x\) in the integral, \(j\text{,}\) is odd.

Example 5.6.1.

Evaluate

\begin{equation*} \displaystyle \int \cos^3 x \sin^2 x \ dx\text{.} \end{equation*}
Solution.

We compute

\begin{align*} \int \cos^3 x \sin^2 x \ dx = \mathstrut \amp \int \cos^2 x \sin^2 x (\cos x) \ dx \\ = \mathstrut \amp \int (1-\sin^2 x)(\sin^2 x) (\cos x) \ dx \\ = \mathstrut \amp \int (\sin^2 x - \sin^4 x) \cos x \ dx\\ = \mathstrut \amp \int (u^2 - u^4) \ du\\ = \mathstrut \amp \frac{1}{3}u^3 - \frac{1}{5}u^5 + C \\ = \mathstrut \amp \frac{1}{3}\sin^3x - \frac{1}{5}\sin^5x + C \end{align*}

where \(u = \sin x\) so that \(du = \cos x \ dx\text{.}\)

In the next example, we discover a strategy that can be successfully applied when there are only even powers of \(\sin x\) and \(\cos x\text{.}\) For integrals of this type, the power-reducing identities

\begin{equation} \sin^2 x = \frac{1}{2} - \frac{1}{2}\cos(2x) = \frac{1-\cos(2x)}{2}\tag{5.15} \end{equation}

and

\begin{equation} \cos^2 x = \frac{1}{2} + \frac{1}{2}\cos(2x) = \frac{1+\cos(2x)}{2}\tag{5.16} \end{equation}

are used.

Example 5.6.2.

Evaluate

\begin{equation*} \displaystyle \int \sin^2 x \ dx. \end{equation*}
Solution.

The trigonometric identity in (5.15) gives

\begin{align*} \int \sin^2 x \ dx = \mathstrut \amp \int \frac{1}{2} - \frac{1}{2}\cos(2x) \ dx\\ = \mathstrut \amp \frac{1}{2}x - \frac{1}{4}\sin(2x) + C. \end{align*}

The general process for integrating products of powers of \(\sin x\) and \(\cos x\) is summarized in the following set of guidelines.

Integrating Products and Powers of \(\sin x\) and \(\cos x\).

To integrate \(\displaystyle \int \cos^j x \sin^k x \ dx\text{,}\) use the following strategies:

  • If \(k\) is odd, rewrite \(\sin^k x = \sin^{k-1}x \sin x\) and use the identity \(\sin^2 x = 1-\cos^2 x\) to rewrite \(\sin^{k-1}x\) in terms of \(\cos x\text{.}\) Integrate using the substitution \(u=\cos x\text{.}\)

  • If \(j\) is odd, rewrite \(\cos^j x = \cos^{j-1}x \cos x\) and use the identity \(\cos^2 x = 1-\sin^2 x\) to rewrite \(\cos^{j-1}x\) in terms of \(\sin x\text{.}\) Integrate using the substitution \(u=\sin x\text{.}\)

  • If both \(j\) and \(k\) are even, use identities (5.15) and (5.16). Then, simplify, and reapply these three strategies as appropriate.

Example 5.6.3.

Evaluate

\begin{equation*} \displaystyle \int \sin^4 x \ dx. \end{equation*}
Solution.

Since the power of \(\sin x\) is even (\(k=4\)), we proceed as follows:

\begin{align*} \int \sin^4x \ dx = \mathstrut \amp \int (\sin^2 x)^2 \ dx\\ = \mathstrut \amp \int \left( \frac{1}{2} - \frac{1}{2}\cos(2x)\right)^2 \ dx\\ = \mathstrut \amp \int \frac{1}{4} - \frac{1}{2}\cos(2x) + \frac{1}{4}\cos^2(2x) \ dx \end{align*}

Then, since \(\cos^2(2x)\) has an even power, we substitute \(\displaystyle \cos^2(2x) = \frac{1}{2} + \frac{1}{2}\cos(4x)\) to see that

\begin{align*} \int \sin^4x \ dx = \mathstrut \amp \int \frac{3}{8} - \frac{1}{2}\cos(2x) + \frac{1}{4}\left( \frac{1}{2} + \frac{1}{2}\cos(4x)\right) \ dx\\ = \mathstrut \amp \int \frac{3}{8} - \frac{1}{2}\cos(2x) + \frac{1}{8}\cos(4x) \ dx\\ = \mathstrut \amp \frac{3}{8}x - \frac{1}{4}\sin(2x) + \frac{1}{32}\sin(4x) + C. \end{align*}

Subsection 5.6.2 Integrating Products of Sines and Cosines of Different Angles

In some areas of physics, such as quantum mechanics, signal processing, and the computation of Fourier series, it is often necessary to integrate products that include \(\sin(ax)\text{,}\) \(\sin(bx)\text{,}\) \(\cos(ax)\text{,}\) and \(\cos(bx)\text{.}\) These integrals are evaluated by applying the following trigonometric identities.

\begin{equation} \sin(ax)\sin(bx) = \frac{1}{2}\cos((a-b)x)-\frac{1}{2}\cos((a+b)x)\tag{5.17} \end{equation}
\begin{equation} \sin(ax)\cos(bx) = \frac{1}{2}\sin((a-b)x)+\frac{1}{2}\sin((a+b)x)\tag{5.18} \end{equation}
\begin{equation} \cos(ax)\cos(bx) = \frac{1}{2}\cos((a-b)x)+\frac{1}{2}\cos((a+b)x)\tag{5.19} \end{equation}

Subsection 5.6.3 Integrating Products and Powers of \(\tan x\) and \(\sec x\)

Let's start by recalling the integrals involving \(\tan x\) and \(\sec x\) we have already learned:

\begin{equation*} \int \sec^2 x \ dx = \tan x + C \end{equation*}
\begin{equation*} \int \sec x \tan x \ dx = \sec x + C \end{equation*}
\begin{equation*} \int \tan x \ dx = \ln|\sec x| + C \end{equation*}
\begin{equation*} \int \sec x \ dx = \ln|\sec x + \tan x| + C \end{equation*}

For most integrals of products and powers of \(\tan x\) and \(\sec x\text{,}\) we rewrite the expression we wish to integrate as the sum or difference of integrals of the form \(\displaystyle \int \tan^j x \sec^2 x \ dx\) or \(\displaystyle \int \sec^j x \tan x \ dx\text{.}\) We can evaluate these new integrals by using substitution.

Example 5.6.4.

Evaluate

\begin{equation*} \displaystyle \int \sec^5 x \tan x \ dx. \end{equation*}
Solution.

We start by rewriting \(\sec^5x \tan x\) as \(\sec^4x \sec x \tan x\text{.}\) Then, we use the substitution \(u=\sec x\) so that \(du = \sec x \tan x \ dx\text{.}\)

\begin{align*} \int \sec^5 x \tan x \ dx = \mathstrut \amp \int \sec^4x (\sec x \tan x) \ dx\\ = \mathstrut \amp \int u^4 \ du\\ = \mathstrut \amp \frac{1}{5}u^5 + C\\ = \mathstrut \amp \frac{1}{5}\sec^5 x + C \end{align*}

Integrating \(\displaystyle \int \tan^k x \sec^j x \ dx\).

  • If \(j\) is even and \(j \geq 2\text{,}\) rewrite \(\sec^j x = \sec^{j-2}x \sec^2 x\) and use the identity \(\sec^2 x = \tan^2x + 1\) to rewrite \(\sec^{j-2}x\) in terms of \(\tan x\text{.}\) Integrate using the substitution \(u=\tan x\text{.}\)

  • If \(k\) is odd and \(j \geq 1\text{,}\) rewrite \(\tan^kx \sec^j x = \tan^{k-1}x \sec^{j-1} x \sec x \tan x\) and use the identity \(\tan^2 x = \sec^2x - 1\) to rewrite \(\tan^{k-1}x\) in terms of \(\sec x\text{.}\) Integrate using the substitution \(u=\sec x\text{.}\)

  • If \(k\) is odd where \(k \geq 3\) and \(j=0\text{,}\) rewrite \(\tan^kx = \tan^{k-2}x\tan^2x = \tan^{k-2}(\sec^2x-1) = \tan^{k-2}x\sec^2x - \tan^{k-2}x\text{.}\) It may then be necessary to repeat this process on the \(\tan^{k-2}x\) term.

  • If \(k\) is even and \(j\) is odd, use \(\tan^2x = \sec^2x-1\) to express \(\tan^kx\) in terms of \(\sec x\text{.}\) Then, use integration by parts to integra odd powers of \(\sec x\text{.}\)

Example 5.6.5.

Evaluate

\begin{equation*} \displaystyle \int \sec^3 x \ dx. \end{equation*}
Solution.

This integral requires integration by parts. Let \(u=\sec x\) and \(dv = \sec^2 x \ dx\text{.}\) These choices make \(du = \sec x \tan x\) and \(v=\tan x\text{.}\) Thus,

\begin{align*} \int \sec^3 x \ dx = \mathstrut \amp \sec x \tan x - \int \tan x \sec x \tan x \ dx\\ = \mathstrut \amp \sec x \tan x- \int \tan^2x \sec x \ dx \\ = \mathstrut \amp \sec x \tan x - \int (\sec^2x-1)\sec x \ dx\\ = \mathstrut \amp \sec x \tan x - \int \sec x \ dx - \int \sec^3 x \ dx\\ = \mathstrut \amp \sec x \tan x + \ln|\sec x + \tan x| - \int \sec^3 x \ dx. \end{align*}

We now have

\begin{equation} \int \sec^3 x = \sec x \tan x +\ln|\sec x + \tan x| - \int \sec^3 x \ dx. \tag{5.20} \end{equation}

Since the integral \(\displaystyle \int \sec^3x \ dx\) has reappeared on the right-hand side, we can solve equation (5.20) for \(\displaystyle \int \sec^3 x \ dx\) by adding it to both sides. In doing so, we obtain

\begin{equation*} 2\int \sec^3x \ dx = \sec x \tan x + \ln|\sec x + \tan x|. \end{equation*}

Dividing by 2, we arrive at

\begin{equation*} \int \sec^3 x \ dx = \frac{1}{2}\sec x \tan x + \frac{1}{2}\ln|\sec x + \tan x| + C. \end{equation*}

Subsection 5.6.4 Summary

Integrating products and powers of many trigonometric functions can be done through strategic use of trigonometric identities and existing integration techniques. Also useful is the ability to integrate products of sines and cosines of different angles. These too are handled by the use of trigonometric identities.
