When \(\theta = 0\) we have \(r=3\sin (2\cdot 0) = 0\text{.}\) The next value for which \(r=0\) is \(\theta = \frac{\pi}{2}\text{.}\) This can be seen by solving the equation \(3\sin (2\theta)=0\) for \(\theta\text{.}\) Therefore the values \(\theta=0\) to \(\theta = \frac{\pi}{2}\) trace out the first petal of the rose. To find the area inside this petal, use equationÂ (9.8) with \(f(\theta=3\sin (2\theta)\text{,}\) \(\alpha=0\text{,}\) and \(\beta=\frac{\pi}{2}\text{:}\)

\begin{align*} A = \mathstrut \amp \frac{1}{2} \int_{\alpha}^{\beta} f(\theta)^2 \ d\theta\\ = \mathstrut \amp \frac{1}{2} \int_0^{\pi/2} [3\sin (2\theta)]^2 \ d\theta\\ = \mathstrut \amp \frac{1}{2} \int_0^{\pi/2} 9\sin^2 (2\theta) \ d\theta\\ = \mathstrut \amp \frac{9}{2} \int_0^{\pi/2} \frac{1-\cos (4\theta)}{2} \ d\theta\\ = \mathstrut \amp \frac{9}{4} \int_0^{\pi/2} 1-\cos (4\theta) \ d\theta\\ = \mathstrut \amp \frac{9}{4} \left( \theta - \frac{\sin(4\theta)}{4} \right) \big\rvert_0^{\pi/2}\\ = \mathstrut \amp \frac{9}{4} \left( \frac{\pi}{2} - \frac{\sin 2\pi}{4} \right) - \frac{9}{4} \left( 0 - \frac{\sin 4(0)}{4} \right)\\ = \mathstrut \amp \frac{9\pi}{8}. \end{align*}
