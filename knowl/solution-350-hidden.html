Since \(\sqrt{9-4x^2} = \sqrt{9-(2x)^2}\text{,}\) we will want to substitute

\begin{equation} 2x=3\sin\theta\tag{5.21} \end{equation}

so that we can factor out the 9 and obtain the expression \(1-\sin^2\theta\text{.}\) Assuming \(2x=3\sin\theta\text{,}\) then \((2x)^2=9\sin^2\theta\) and \(2 \ dx = 3\cos\theta \ d\theta\text{.}\) Then,

\begin{align*} \int \frac{1}{\sqrt{9-4x^2}} \ dx = \mathstrut \amp \int \frac{1}{\sqrt{9-(2x)^2}} \ dx\\ = \mathstrut \amp \int \frac{(3/2)\cos\theta}{\sqrt{9-9\sin^2\theta}} \ d\theta\\ = \mathstrut \amp \frac{3}{2}\int \frac{\cos\theta}{\sqrt{9(1-\sin^2\theta)}} \ d\theta\\ = \mathstrut \amp \frac{3}{2}\int \frac{\cos\theta}{3\sqrt{\cos^2\theta}} \ d\theta\\ = \mathstrut \amp \frac{3}{2}\int \frac{\cos\theta}{3\cos\theta} \ d\theta\\ = \mathstrut \amp \frac{3}{6}\int 1 \ d\theta\\ = \mathstrut \amp \frac{1}{2}\theta + C. \end{align*}

We can solve for \(\theta\) in Equation (5.21) to find

\begin{equation*} \int \frac{1}{\sqrt{9-4x^2}} \ dx = \frac{1}{2}\arcsin \left( \frac{2x}{3} \right) + C. \end{equation*}
