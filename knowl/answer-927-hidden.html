1. Remember that WolframAlpha can aid in sketching the graph of each of these.

  2. The points of intersection occur when \(\theta = \frac{\pi}{6}\) and \(\theta = \frac{5\pi}{6}\text{.}\)

  3. The area inside the circle \(r=6\sin\theta\) is given by

    \begin{equation*} \frac{1}{2}\int_{\pi/6}^{5\pi/6} [6\sin\theta]^2 \ d\theta = \frac{9\sqrt{3}}{2} + 6\pi\text{.} \end{equation*}

    The area inside the cardioid \(r=2+2\sin\theta\) is given by

    \begin{equation*} \frac{1}{2}\int_{\pi/6}^{5\pi/6} [2+2\sin\theta]^2 \ d\theta = \frac{9\sqrt{3}}{2}+2\pi\text{.} \end{equation*}

    Thus the area bounded between these two curves is the difference, \(4\pi\text{.}\)

  4. The reason why this point did not show up as a solution is because the origin is on both graphs but for different values of \(\theta\text{.}\) For example, for the cardioid we get

    \begin{align*} 2+2\sin\theta = \mathstrut \amp 0\\ \sin \theta = \mathstrut \amp -1 \end{align*}

    so the values for \(\theta\) that solve this equation are \(\theta = \frac{3\pi}{2}+2n\pi\text{,}\) where \(n\) is any integer. For the circle we get

    \begin{equation*} 6\sin\theta = 0. \end{equation*}

    The solutions to this equation are of the form \(\theta = n\pi\) for any integer value of \(n\text{.}\) These two solution sets have no points in common. Regardless of this fact, the curves intersect at the origin.
