Preview Activity 5.6.1.

  1. Evaluate \(\displaystyle \int \cos^3 x \sin x \ dx\) by using the \(u\)-substitution \(u=\cos x\text{.}\)

  2. Evaluate \(\displaystyle \int \sin^4 x \cos x \ dx\text{.}\)

  3. Evaluate \(\displaystyle \int \sin^3 x \ dx\) by rewriting \(\sin^3 x = \sin^2 x \sin x\) and using the identity \(\sin^2 x = 1-\cos^2 x\) followed by the \(u\)-substitution \(u=\cos x\text{.}\)

  4. Evaluate \(\displaystyle \int \cos^2 x \sin^3 x \ dx\) by rewriting \(\sin^3 x = \sin^2 x \sin x\) and using the identity \(\sin^2 x = 1-\cos^2 x\) followed by the \(u\)-substitution \(u=\cos x\text{.}\)
