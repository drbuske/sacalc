Solution 5.6.3.1.

Since the power of \(\sin x\) is even (\(k=4\)), we proceed as follows:

\begin{align*} \int \sin^4x \ dx = \mathstrut \amp \int (\sin^2 x)^2 \ dx\\ = \mathstrut \amp \int \left( \frac{1}{2} - \frac{1}{2}\cos(2x)\right)^2 \ dx\\ = \mathstrut \amp \int \frac{1}{4} - \frac{1}{2}\cos(2x) + \frac{1}{4}\cos^2(2x) \ dx \end{align*}

Then, since \(\cos^2(2x)\) has an even power, we substitute \(\displaystyle \cos^2(2x) = \frac{1}{2} + \frac{1}{2}\cos(4x)\) to see that

\begin{align*} \int \sin^4x \ dx = \mathstrut \amp \int \frac{3}{8} - \frac{1}{2}\cos(2x) + \frac{1}{4}\left( \frac{1}{2} + \frac{1}{2}\cos(4x)\right) \ dx\\ = \mathstrut \amp \int \frac{3}{8} - \frac{1}{2}\cos(2x) + \frac{1}{8}\cos(4x) \ dx\\ = \mathstrut \amp \frac{3}{8}x - \frac{1}{4}\sin(2x) + \frac{1}{32}\sin(4x) + C. \end{align*}
