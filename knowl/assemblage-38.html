Integrating \(\displaystyle \int \tan^k x \sec^j x \ dx\).

  • If \(j\) is even and \(j \geq 2\text{,}\) rewrite \(\sec^j x = \sec^{j-2}x \sec^2 x\) and use the identity \(\sec^2 x = \tan^2x + 1\) to rewrite \(\sec^{j-2}x\) in terms of \(\tan x\text{.}\) Integrate using the substitution \(u=\tan x\text{.}\)

  • If \(k\) is odd and \(j \geq 1\text{,}\) rewrite \(\tan^kx \sec^j x = \tan^{k-1}x \sec^{j-1} x \sec x \tan x\) and use the identity \(\tan^2 x = \sec^2x - 1\) to rewrite \(\tan^{k-1}x\) in terms of \(\sec x\text{.}\) Integrate using the substitution \(u=\sec x\text{.}\)

  • If \(k\) is odd where \(k \geq 3\) and \(j=0\text{,}\) rewrite \(\tan^kx = \tan^{k-2}x\tan^2x = \tan^{k-2}(\sec^2x-1) = \tan^{k-2}x\sec^2x - \tan^{k-2}x\text{.}\) It may then be necessary to repeat this process on the \(\tan^{k-2}x\) term.

  • If \(k\) is even and \(j\) is odd, use \(\tan^2x = \sec^2x-1\) to express \(\tan^kx\) in terms of \(\sec x\text{.}\) Then, use integration by parts to integra odd powers of \(\sec x\text{.}\)
