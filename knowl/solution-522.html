Solution 8.4.6.1.

We have

\begin{equation*} \arctan(x) = \sum_{k=0}^{\infty} (-1)^k \frac{x^{2k+1}}{(2k+1)}\text{.} \end{equation*}

Since \(\arctan(1) = \frac{\pi}{4}\text{,}\) the series gives us a pretty formula for \(\pi\text{:}\)

\begin{align*} \frac{\pi}{4} \amp = \sum_{k=0}^{\infty} \frac{(-1)^k}{2k+1}\\ \amp = 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots \end{align*}

So

\begin{equation*} \pi = 4 \left( 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots \right)\text{.} \end{equation*}

Unfortunately, this series is not very useful for computing \(\pi\) because it converges so slowly. If we approximate the series by its \(n\)th partial sum, then the alternating series test tells us that the error is bounded by the first term we drop. To guarantee that we have 2 decimals of \(\pi\) correct, we need to sum the first 100 terms! That is,

\begin{equation*} \pi \approx 4\sum_{k=0}^{99} \frac{(-1)^k}{2k+1} = 4\left( 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots - \frac{1}{199} \right)\text{.} \end{equation*}
