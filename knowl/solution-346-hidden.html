We start by rewriting \(\sec^5x \tan x\) as \(\sec^4x \sec x \tan x\text{.}\) Then, we use the substitution \(u=\sec x\) so that \(du = \sec x \tan x \ dx\text{.}\)

\begin{align*} \int \sec^5 x \tan x \ dx = \mathstrut \amp \int \sec^4x (\sec x \tan x) \ dx\\ = \mathstrut \amp \int u^4 \ du\\ = \mathstrut \amp \frac{1}{5}u^5 + C\\ = \mathstrut \amp \frac{1}{5}\sec^5 x + C \end{align*}
