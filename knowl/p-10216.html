In rectangular coordinates, the arc length of a parameterized curve

\begin{equation*} x=x(t), \ y=y(t), \ a \leq t \leq b \end{equation*}

is given by

\begin{equation*} L = \int_a^b \sqrt{\left( \frac{dx}{dt}\right)^2 + \left( \frac{dy}{dt}\right)^2} \ dt\text{.} \end{equation*}

In polar coordinates we define the curve by the equation \(r=f(\theta)\text{,}\) where \(\alpha \leq \theta \leq \beta\text{.}\) In order to adapt the arc length formula for a polar curve, we use the equations

\begin{align*} x = \mathstrut \amp r\cos \theta = f(\theta)\cos\theta\\ y = \mathstrut \amp r\sin\theta = f(\theta)\sin\theta \end{align*}

and we replace the parameter \(t\) by \(\theta\text{.}\) Then

\begin{align*} \frac{dx}{d\theta} = \mathstrut \amp f'(\theta) \cos \theta - f(\theta)\sin \theta\\ \frac{dy}{d\theta} = \mathstrut \amp f'(\theta)\sin\theta + f(\theta)\cos\theta. \end{align*}

We replace \(dt\) by \(d\theta\text{,}\) and the lower and upper limits of integration are \(\alpha\) and \(\beta\) respectively. Then the arc length formula becomes

\begin{align*} L = \mathstrut \amp \int_a^b \sqrt{ \left( \frac{dx}{dt} \right)^2 + \left( \frac{dy}{dt} \right)^2} \ dt\\ = \mathstrut \amp \int_{\alpha}^{\beta} \sqrt{ \left( \frac{dx}{d\theta} \right)^2 + \left( \frac{dy}{d\theta} \right)^2} \ d\theta\\ = \mathstrut \amp \int_{\alpha}^{\beta} \sqrt{ [f'(\theta)\cos\theta - f(\theta)\sin\theta ]^2 + [f'(\theta)\sin\theta + f(\theta)\cos\theta ]^2} \ d\theta\\ = \mathstrut \amp \int_{\alpha}^{\beta} \sqrt{ f'(\theta)^2 [\cos^2\theta + \sin^2 \theta] + f(\theta)^2 [\cos^2\theta + \sin^2\theta]} \ d\theta\\ = \mathstrut \amp \int_{\alpha}^{\beta} \sqrt{ f'(\theta)^2 + f(\theta)^2} \ d\theta\\ = \mathstrut \amp \int_{\alpha}^{\beta} \sqrt{ r^2 + \left( \frac{dr}{d\theta} \right)^2 } \ d\theta. \end{align*}
