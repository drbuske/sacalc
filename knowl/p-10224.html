  1. Converting to rectangular coordinates gives \(x^2 + \left( y-\frac{3}{2}\right)^2 = \left( \frac{3}{2}\right)^2\text{.}\) This is a circle of radius \(\frac{3}{2}\) centered at \(\left( 0, \frac{3}{2} \right)\text{.}\)

  2. \(L= \int_0^{\pi} \sqrt{ (3\sin\theta)^2 + (3\cos\theta)^2} \ d\theta = \int_0^{\pi} 3\sqrt{\sin^2\theta + \cos^2\theta} \ d\theta = 3\pi\) which is exactly the perimeter one would expect since the curve is a circle of diameter \(3\text{.}\)

  3. \(A = \frac{1}{2} \int_0^{\pi} (3\sin\theta)^2 \ d\theta = \frac{9\pi}{4} = \pi\left(\frac{3}{2}\right)^2\) which is exactly the area one would expect since the curve is a circle of radius \(\frac{3}{2}\text{.}\)
