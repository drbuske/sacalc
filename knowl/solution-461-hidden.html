1. \(\lim_{k \rightarrow \infty} \frac{1}{k} = 0\text{,}\) so the divergence test is inconclusive. It's true that this series diverges, but we can't show it using the divergence test.

  2. \(\lim_{k \rightarrow \infty} \frac{k^2}{k+1} = \infty\text{,}\) which is not zero, so the divergence test tells us that this series diverges.

  3. \(\lim{k \rightarrow \infty} \sin k\) does not exist. Since it is not zero, the divergence test tells us that the series diverges.

  4. For all whole numbers \(k, \sin(\pi k)=0\text{,}\) so \(\lim_{k \rightarrow \infty} \sin (\pi k) = 0\) and the divergence test is inconclusive.
