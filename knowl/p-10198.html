Consider a curve defined by the function \(r=f(\theta)\text{,}\) where \(\alpha \leq \theta \leq \beta\text{.}\) Our first step is to partition the interval \([\alpha, \beta]\) into \(n\) equal-width subintervals. The width of each subinterval is given by the formula \((\beta-\alpha)/n\text{,}\) and the \(i\)th partition point \(\theta_i\) is given by the formula \(\theta_i = \alpha + i\Delta \theta\text{.}\) Each partition point \(\theta = \theta_i\) defines a line with slope \(\tan \theta_i\) passing through the pole as shown in FigureÂ 9.4.2.
