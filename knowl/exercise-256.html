Exercise 5. Average Value of a Product of Trigonometric Functions.

  1. Graph the function \(f(x)=\sin^2x\cos^3x\) on the interval \([-\pi,\pi]\text{.}\)

  2. Evaluate \(\displaystyle \int \sin^2x \cos^3x \ dx\text{.}\)

  3. Using your answer to 5.6.5.5.b, calculate the average value of the function \(f(x)=\sin^2x\cos^3x\) over the interval \([-\pi,\pi]\text{.}\)

  4. Use technology to directly compute the value of the definite integral \(\displaystyle \frac{1}{2\pi}\int_{-\pi}^{\pi} \sin^2x \cos^3x \ dx\text{.}\)

  5. Calculate the average value of the function \(f(x)=\sin^2x\cos^3x\) over the interval \([-\pi/2,\pi/2]\text{.}\)
