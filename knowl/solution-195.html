Solution 3.1.4.2.1.

SOLUTION

We find inflection points by determining where \(f''(x)\) changes sign. \(f'(x) = 3\cdot 4x^{3}+17\cdot 3x^{2}-9\cdot 2x\text{,}\) so \(f''(x) = 3\cdot 4\cdot 3x^{2}+17\cdot 3\cdot 2x-18 = 6(6 x - 1)(x + 3)\text{.}\) So we see that \(f''(x)\) changes sign at \(x = 1/6\) and \(x = -3\text{.}\)
