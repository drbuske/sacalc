Answer 7.5.4.7.1.

  1. \(\displaystyle 10^d - 10^{d-1}\)

  2. \(\displaystyle h(d) < 9^d\)

  3. \(\displaystyle m(d) = 10^{d-1} + 1\)

  4. Group the terms in the deleted harmonic series together by the number of digits. Then \(h(d)\) bounds the number of terms, and each term is as most \(1/m(d)\text{.}\)

  5. \(\sum_{d=1}^{\infty} \frac{h(d)}{m(d)} \leq \sum_{d=1}^{\infty} \frac{9^d}{10^{d-1}} \leq 90\) (i.e. compare to a geometric series). One can actually use comparison to estimate the value to smaller than 80. The actual value is smaller than 23.
