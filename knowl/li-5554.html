Item b.

To evaluate \(\int_0^1 \sin(x^2) \ dx\text{,}\) we evaluate an antiderivative such as the one found in (a) at both 1 and 0 and subtract. So

\begin{align*} \int_0^1 \sin(x^2) \ dx \amp = \sum_{k=0}^{\infty} (-1)^k \frac{x^{4k+3}}{(2k+1)!(4k+3)} \bigg\|_0^1\\ \amp = \sum_{k=0}^{\infty} (-1)^k \frac{1}{(2k+1)!(4k+3)} \end{align*}

The resulting series is an alternating series, so we can find a value of \(n\) that makes the \(n\)th partial sum of the series approximate the sum of the series to three decimal places by determining a value of \(n\) such that

\begin{equation*} a_{n+1} = \frac{1}{(2(n+1)+1)! (4(n+1)+3)} < 0.001\text{.} \end{equation*}

This is not an inequality that we can solve algebraically, so we use trial and error. Doing so shows, perhaps surprisingly, that \(n=1\) will work.
