(a) To determine if the particles collide, we check whether they are ever at the same point at the same time. We first set the two \(x\)-coordinates equal to each other: \(4 t - 4 = 3 t\text{,}\) so that \(t = 4\text{.}\) Then, when \(t=4\text{,}\) we need to determine if the \(y\)-coordinates are also equal. \(y_A(4) = 12 - 3 = 9\text{,}\) and \(y_B(4) = 4^2 - 2\cdot 4 - 1 = 7\text{.}\) These are not equal, so the particles do not collide. (b) For the particles to collide, we continue as in (a), but without the given value of \(k\text{.}\) For the \(x\)-coordinates of the trajectories to be the same we have the same condition as before, \(t = 4\text{.}\) Then, with this value of \(t\text{,}\) we get \(y_A(4) = 3\cdot 4 - k = 12 - k\text{,}\) and \(y_B(4) = 4^2 - 2\cdot 4 - 1 = 7\text{.}\) Setting these equal, we get \(k = 5\text{.}\) (c) The speeds of the particles are given by \(s = \sqrt{x'(t)^2 + y'(t)^2}\text{.}\) For particle \(A\text{,}\) the speed is \(s_A = \sqrt{4^2 + 3^2} = \sqrt{25}\text{.}\) For particle \(B\text{,}\) the speed is \(s_B = \sqrt{3^2 + 4(t - 1)^2}\text{.}\) Evaluating this latter at \(t = 4\text{,}\) we find that particle B is faster.
