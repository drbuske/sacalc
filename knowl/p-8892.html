  1. If we evaluate \(f\) at \(x=1\) we obtain the series

    \begin{equation*} \sum_{k=0}^{\infty} \frac{1}{2^k} \end{equation*}

    which is a geometric series with ratio \(\frac{1}{2}\text{.}\) So we can sum this series and find that \(f(1)=\frac{1}{1-\frac{1}{2}} = 2\text{.}\)

    Similarly,

    \begin{equation*} f \left( \frac{3}{2} \right) = \sum_{k=0}^{\infty} \left( \frac{3}{4} \right)^k = \frac{1}{1-\frac{3}{4}} = 4\text{.} \end{equation*}

  2. In general, \(f(x)\) is a geometric series with ratio \(\frac{x}{2}\text{,}\) so

    \begin{equation*} f(x) = \sum_{k=0}^{\infty} \left( \frac{x}{2} \right)^k = \frac{1}{1-\frac{x}{2}} = \frac{2}{2-x} \end{equation*}

    provided that \(-1 < \frac{x}{2} < 1\) (which ensures that the ratio is less than \(1\) in absolute value).

  3. The power series that defines \(f(x) = \frac{2}{2-x}\) converges for \(-2 < x < 2\text{.}\)
