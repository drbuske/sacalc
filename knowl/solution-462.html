Solution 7.5.1.1.

  1. When \(k\) is very large, the constant 2 in the numerator is negligible when compared to \(k\text{,}\) and the constant 1 in the denominator is negligible when compared to \(k^4\text{.}\) So for large \(k\text{,}\) the numerator looks like \(k\) and the denominator looks like \(k^4\text{.}\) Thus, the fraction \(\frac{k-2}{k^4+1}\) looks like \(\frac{k}{k^4}\) for large \(k\text{.}\)

  2. We know that \(\sum \frac{1}{k^3}\) is a \(p\)-series with \(p=3\text{,}\) so it is a convergent series. Thus, we expect that \(\sum \frac{k-2}{k^4+1}\) will converge too.

  3. Since a fraction increases if we make the numerator larger or if we make the denominator smaller, we can do the following:

    \begin{equation*} 0 \leq \frac{k-2}{k^4+1} \leq \frac{k}{k^4+1} \leq \frac{k}{k^4} = \frac{1}{k^3} \end{equation*}

    for all \(k \geq 1\text{.}\)

  4. Each term in the sum of \(S_n\) is less than the corresponding term in the sum \(T_n\text{.}\)

  5. Yes. The partial sums \(\{T_n\}\) converge since the series \(\sum_{k=1}^{\infty} \frac{1}{k^3}\) converges by virtue of being a \(p\)-series with \(p>1\text{.}\)

  6. The sequence \(\{S_n\}\) is monotone increasing since for \(n \geq 2\text{,}\) the terms being added are nonnegative. The sequence \(\{S_n\}\) is bounded since \(S_n \leq T_n \leq T\text{.}\)
