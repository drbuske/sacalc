If \(\lim_{n \rightarrow \infty} s_n = L\text{,}\) then from some point on, the terms are bounded between \(L-1\) and \(L+1\text{.}\) So, a convergent sequence is bounded. However, a bounded sequence need not be convergent. For example, \(s_n = 1+(-1)^n\) is bounded between 0 and 2 and does not converge. If we avoid the "oscillation", however, we can ensure convergence.
