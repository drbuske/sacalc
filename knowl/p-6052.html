The general trigonometric substitution process for evaluating integrals involving \(\sqrt{a^2-x^2}\) makes use of the substitution \(x=a\sin\theta\) so that \(dx = a\cos\theta\text{.}\) The domain of \(\sqrt{a^2-x^2}\) is \([-a,a]\) so that \(-a \leq x \leq a\) or \(-1 \leq \frac{x}{a} \leq 1\text{.}\) Since the range of \(\sin x\) over \([-\pi/2,\pi/2]\) is \([-1,1]\text{,}\) there is a unique angle \(\theta\) satisfying \(-\pi/2 \leq \theta \leq \pi/2\) so that \(\sin\theta = \frac{x}{a}\text{,}\) or equivalently, so that \(x=a\sin\theta\text{.}\) If we substitute \(x=a\sin\theta\) into \(\sqrt{a^2-x^2}\text{,}\) we get

\begin{align*} \sqrt{a^2-x^2} = \mathstrut \amp \sqrt{a^2-(a\sin\theta)^2}\\ = \mathstrut \amp \sqrt{a^2-a^2\sin^2\theta}\\ = \mathstrut \amp \sqrt{a^2(1-\sin^2\theta)}\\ = \mathstrut \amp \sqrt{a^2\cos^2\theta}\\ = \mathstrut \amp |a\cos\theta|\\ = \mathstrut \amp a\cos\theta \end{align*}

where \(|a\cos\theta| = a\cos\theta\) since \(\cos\theta \geq 0\) on \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\text{.}\)
