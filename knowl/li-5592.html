To create a graph of this curve, first set up a table of values. Since the independent variable in both \(x(t)\) and \(y(t)\) is \(t\text{,}\) let \(t\) appear in the first column. Then \(x(t)\) and \(y(t)\) will appear in the second and third columns of Table 9.1.7.

The second and third columns in Table 9.1.7 provide a set of points to be plotted. The graph of these points appears in Figure 9.1.8. The arrows on the graph indicate the orientation of the graph, that is, the direction that a point moves on the graph as \(t\) varies from −3 to 2.
