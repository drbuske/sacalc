\begin{equation*} {dx \over dt} = -\sin\!\left(t\right), {dy \over dt} = 0.25\cos\!\left(\frac{t}{4}\right). \end{equation*}
The speed is
\begin{equation*} v = \sqrt{\sin^{2}\!\left(t\right)+\frac{\cos^{2}\!\left(\frac{t}{4}\right)}{16}}\ {\textstyle\frac{\rm\mathstrut ft}{\rm\mathstrut s}}. \end{equation*}
Thus, \(v=0\) when \(-\sin\!\left(t\right) = 0.25\cos\!\left(\frac{t}{4}\right) =0\text{,}\) and so the particle stops first when \(t = 6.28319\ {\rm s}\text{,}\) or, with \(n\) being any odd integer, when \(t = 2n\pi \ {\rm s}\text{.}\)
