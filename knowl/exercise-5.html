For each sequence, decide if it is bounded and if it is monotone. (It may help you to list out the first few terms of the sequence.)

  2. \(\displaystyle b_n = 4\sin (n)\)

  3. \(\displaystyle c_n = n(-1)^n\)

  4. \(\displaystyle d_n = n^2+n\)
