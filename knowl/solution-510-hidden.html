(a)\(x\) is the first degree approximation of \(f(x)=\sin\!\left(x\right)\text{;}\) it is also the second degree approximation, since the next term in the Taylor expansion is 0. \(P_1(x) = x\) is an overestimate for \((0,1]\) and an underestimate for \([-1,0)\text{.}\) (This can be seen easily from a graph.) (b) Using the second degree error bound, if \(|f^{(3)}(x)| \leq M\) for \(-1 \leq x \leq 1\text{,}\) then \(|E_{2}|\leq \frac{M\cdot |x|^{3}}{{3}!} \leq \frac{M}{6}.\) For what value of \(M\) is \(|f^{(3)}(x)| \leq M\) for \(-1 \leq x \leq 1\text{?}\) Well, \(|f^{(3)}(x)|=|-\cos\!\left(x\right)| \leq 1\text{.}\) So \(|E_{2}| \leq \frac{1}{6}.\)(c) As before, we note that \(x+\frac{-x^{3}}{3!}+\frac{x^{5}}{5!}\) is the 5th degree approximation of \(f(x)=\sin\!\left(x\right)\text{;}\) it is also the 6th degree approximation, since the next term in the Taylor expansion is 0. This is an overestimate for \((0,1]\) and an underestimate for \([-1,0)\text{.}\) (This also can be seen easily from a graph.) (d) Again, similar to (b), our error bound is \(|E_{6}|\le \frac{M\cdot |x|^{7}}{(7)!} \le \frac{M}{5040},\) where \(|f^{(7)}(x)| \leq M\) for \(-1 \leq x \leq 1\text{.}\) Then a good value for \(M\) is \(M = 1\text{,}\) so \(|E_{6}| \le \frac{1}{5040}\text{.}\)
