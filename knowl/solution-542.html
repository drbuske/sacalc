Solution 9.2.6.5.1.

(a) The curve is a spiral as shown in the figure below.
The point \((x(3.5),y(3.5)) = (-3.2776,-1.22774)\) is shown on the graph. As \(t\) increases, a particle is moving counterclockwise around the spiral, so at this point it is moving down and to the right. (b) We can estimate the speed by finding the position at \(t=3.51\text{:}\) \((x(3.51),y(3.51)) = (-3.27449,-1.26406)\text{,}\) so the speed is approximately speed\(\approx \sqrt{\left({(-3.27449 + 3.2776)\over 0.01}\right)^2 + \left({(-1.26406 + 1.22774)\over 0.01}\right)^2} = 3.64529\text{.}\) (c) Finally, the derivative of the particle's position gives its velocity: \((x'(t),y'(t)) = (\cos\!\left(t\right)-t\sin\!\left(t\right),\sin\!\left(t\right)+t\cos\!\left(t\right))\text{,}\) so \((x'(3.5),y'(3.5)) = (0.291285,-3.62838))\text{,}\) and the speed is actually speed\(= \sqrt{(0.291285)^2 + (-3.62838)^2} = 3.64005\text{.}\)
