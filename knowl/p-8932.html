Again we apply the ratio test. This time we use \(a_k = (k!)^2x^k\text{.}\) For \(x \neq 0\text{,}\) we see that

\begin{align*} \lim_{k \rightarrow \infty} \frac{ \left| a_{k+1} \right|}{\left| a_k \right|} \amp = \lim_{k \rightarrow \infty} \frac{ \left| [(k+1)!]^2x^{k+1} \right|}{\left| (k!)^2x^k \right|}\\ \amp = \lim_{k \rightarrow \infty} \left| (k+1)^2x\right|\\ \amp = \left| x \right| \lim_{k \rightarrow \infty} (k+1)^2\\ \amp = \infty \end{align*}

Therefore, the series diverges for all \(x \neq 0\text{.}\) Since the series is centered at \(x=0\text{,}\) it must converge there so the series converges only for \(x=0\text{.}\) The interval of convergence is the single value \(x=0\text{.}\)
