Activity 7.3.3.

For each series below, find the partial sum for the series (choices are given). Then, determine if the series is convergent or divergent.

  1. \(\displaystyle \displaystyle \sum_{k=1}^{\infty} e^{\frac{1}{k}} - e^{\frac{1}{k+1}}\)

    1. \(\displaystyle \displaystyle S_n = e-e^{\frac{1}{n}}\)

    2. \(\displaystyle \displaystyle S_n = e-e^{\frac{1}{n+1}}\)

    3. \(\displaystyle \displaystyle S_n = e^{\frac{1}{n}}\)

    4. \(\displaystyle \displaystyle S_n = e+e^{\frac{1}{n+1}}\)

    5. \(\displaystyle \displaystyle S_n = e\)

  2. \(\displaystyle \displaystyle \sum_{k=1}^{\infty} 3(2)^{k-1}\)

    1. \(\displaystyle \displaystyle S_n = -3(2^n-1)\)

    2. \(\displaystyle \displaystyle S_n = 6\)

    3. \(\displaystyle \displaystyle S_n = 3(2^n-1)\)

    4. \(\displaystyle \displaystyle S_n = 3(2)^n\)

    5. \(\displaystyle \displaystyle S_n = \frac{1}{6}\)

  3. \(\displaystyle \displaystyle \sum_{k=1}^{\infty} \ln \left( \frac{k+1}{k} \right)\)

    1. \(\displaystyle \displaystyle S_n = \ln \left( \frac{n}{n+1} \right)\)

    2. \(\displaystyle \displaystyle S_n = \ln \left( \frac{n}{n-1} \right)\)

    3. \(\displaystyle \displaystyle S_n = \ln n\)

    4. \(\displaystyle \displaystyle S_n = \ln (n+1)\)

    5. \(\displaystyle \displaystyle S_n = \ln (n-1)\)

  4. \(\displaystyle \displaystyle \sum_{k=1}^{\infty} \frac{1}{(2k+1)(2k-1)}\)

    1. \(\displaystyle \displaystyle S_n = \frac{2}{2n+1}\)

    2. \(\displaystyle \displaystyle S_n = \frac{n}{2n+1}\)

    3. \(\displaystyle \displaystyle S_n = \frac{n}{2(2n+1)}\)

    4. \(\displaystyle \displaystyle S_n = \frac{2n+1}{n}\)

    5. \(\displaystyle \displaystyle S_n = \frac{2n+1}{2n}\)
