The radius of a shell is given by \(x+1\text{,}\) so the volume of the solid is given by

\begin{align*} V = \mathstrut \amp \int_1^2 2\pi (x+1)f(x) \ dx\\ = \mathstrut \amp \int_1^2 2\pi (x+1) x \ dx\\ = \mathstrut \amp 2\pi \int_1^2 x^2 + x \ dx\\ = \mathstrut \amp \frac{23\pi}{3} {\text{ cubic units}}. \end{align*}
