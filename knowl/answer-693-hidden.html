1. Note that

    \begin{equation*} \frac{e^{-k}}{k^2} \leq \frac{1}{k^2} \end{equation*}

    for all \(k \geq 1\text{.}\) This means that \(\sum_{k=1}^{\infty} \frac{e^{-k}}{k^2}\) converges since \(\sum_{k=1}^{\infty} \frac{1}{k^2}\) converges.

  2. Define \(a_k = \frac{k-1}{k^2+3}\text{.}\) Choose \(b_k = \frac{k}{k^3} = \frac{1}{k^2}\) so that both series are positive for \(k \geq 2\text{.}\) Then,

    \begin{equation*} a_k = \frac{k-1}{k^3+3} \leq \frac{k}{k^3} = b_k \end{equation*}

    so that \(a_k \leq b_k\) for \(k \geq 2\text{.}\) Since \(\sum b_k\) converges, so does \(\sum a_k\text{.}\)

  3. Define \(a_k = \frac{6k^2+1}{2k^3-1}\) and choose \(b_k = \frac{6k^2}{2k^3} = \frac{3}{k}\text{.}\) Both \(a_k\) and \(b_k\) are positive for \(k \geq 1\text{.}\) Also, \(a_k \geq b_k\) for all \(k\text{.}\) Since \(\sum b_k\) diverges, \(\sum a_k\) also diverges.
