The polar representation of a point is not unique. For example, the polar coordinates \(\left( 2, \frac{\pi}{3} \right)\) and \(\left( 2, \frac{7\pi}{3}\right)\) both represent the point \((1,\sqrt{3})\) in the rectangular system. Also, the value of \(r\) can be negative. Therefore, the point with polar coordinates \(\left(-2,\frac{4\pi}{3}\right)\) also represents the point \((1,\sqrt{3})\) in the rectangular system. Every point in the plane has an infinite number of representations in polar coordinates. However, each point in the plane has only one representation in the rectangular coordinate system.
