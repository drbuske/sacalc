We start by completing the square in the denominator to find

\begin{align*} \int \frac{2x+7}{x^2+6x+18} \ dx = \mathstrut \amp \int \frac{2x+7}{(x+3)^2+9} \ dx\\ = \mathstrut \amp \int \frac{2(u-3)+7}{u^2+9} \ du\\ = \mathstrut \amp \int \frac{2u}{u^2+9} \ du + \int \frac{4}{u^2+9} \ du \end{align*}

where the substitution \(u=x+3\) has been employed. For the first of these integrals, use the substitution \(w=u^2+9\) and \(dw = 2u \ du\text{.}\) Then,

\begin{align*} \int \frac{2u}{u^2+9} \ du = \mathstrut \amp \int \frac{1}{w} \ dw\\ = \mathstrut \amp \ln |w| + C_1\\ = \mathstrut \amp \ln |(x+3)^2 + 9| + C_1 \end{align*}

For the second integral, \(\int \frac{4}{u^2+9} \ du\text{,}\) we can use a trigonometric substitution or we can factor out the 9, use the substitution \(v=w/3\) and \(dv = \frac{1}{3} \ dw\) and use the antidifferentiation rule \(\int \frac{1}{1+u^2} \ du = \arctan(u)+C\text{.}\) This latter technique gives

\begin{align*} \int \frac{4}{u^2+9} \ du = \mathstrut \amp \frac{4}{9} \int \frac{1}{(u/3)^2+1} \ du\\ = \mathstrut \amp \frac{4}{3} \int \frac{1}{v^2+1} \ dv\\ = \mathstrut \amp \frac{4}{3} \arctan(v) + C_2\\ = \mathstrut \amp \frac{4}{3} \arctan\left( \frac{x+3}{3}\right) + C_2 \end{align*}

Therefore,

\begin{equation*} \int \frac{2x+7}{x^2+6x+18} \ dx = \ln |(x+3)^2+9| + \frac{4}{3}\arctan \left( \frac{x+3}{3} \right) + C\text{.} \end{equation*}
