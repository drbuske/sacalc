Solution 9.3.5.2.1.

A. With \(x={5\over 2}\) and \(y=-{5\sqrt{3}\over 2}\text{,}\) we can find \(r\) from \(r=\sqrt{x^2+y^2}=5\text{.}\) We can find \(\theta\) from \(\tan \theta = y/x = -\sqrt{3}\text{.}\) Thus, \(\theta = -\pi/3\text{.}\) Since \(({5\over 2},-{5\sqrt{3}\over 2})\) is in the fourth quadrant the correct angle must be \(360^\circ\) (\(=2\pi\)) from this. The polar coordinates are \((5,5\pi/3)\text{.}\) B. With \(x={5\sqrt{3}\over 2}\) and \(y=-{5\over 2}\text{,}\) we can find \(r\) from \(r=\sqrt{x^2+y^2}=5\text{.}\) We can find \(\theta\) from \(\tan \theta = y/x = -{1\over\sqrt{3}}\text{.}\) Thus, \(\theta = -\pi/6\text{.}\) Since \(({5\sqrt{3}\over 2},-{5\over 2})\) is in the fourth quadrant the correct angle must be \(360^\circ\) (\(=2\pi\)) from this. The polar coordinates are \((5,11\pi/6)\text{.}\)
