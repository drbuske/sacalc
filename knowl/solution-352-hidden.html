Set

\begin{equation} x=2\tan\theta\tag{5.23} \end{equation}

so that \(dx = 2\sec^2\theta \ d\theta\text{.}\) Since \(x^2=4\tan^2\theta\text{,}\) we use the identity \(1+\tan^2\theta = \sec^2\theta\) to get

\begin{align*} \int \frac{1}{x^2\sqrt{4+x^2}} \ dx = \mathstrut \amp \int \frac{2\sec^2\theta}{4\tan^2\theta \sqrt{4+4\tan^2\theta}} \ d\theta\\ = \mathstrut \amp \int \frac{2sec^2\theta}{4\tan^2\theta \cdot 2\sec\theta} \ d\theta\\ = \mathstrut \amp \frac{1}{4} \int \frac{\sec\theta}{\tan^2\theta} \ d\theta\\ = \mathstrut \amp \frac{1}{4} \int \frac{1}{\cos\theta} \frac{\cos^2\theta}{\sin^2\theta} \ d\theta\\ = \mathstrut \amp \frac{1}{4} \int \frac{\cos\theta}{\sin^2\theta} \ d\theta. \end{align*}

This last integral can be evaluated via yet another substitution. This time, a \(u\)-substitution in which we set \(u=\sin\theta\) so that \(du = \cos\theta \ d\theta\) will be our path forward. Doing so gives

\begin{align*} \int \frac{1}{x^2\sqrt{4+x^2}} \ dx = \mathstrut \amp \frac{1}{4} \int \frac{\cos\theta}{\sin^2\theta} \ d\theta\\ = \mathstrut \amp \frac{1}{4} \int \frac{1}{u^2} \ du\\ = \mathstrut \amp -\frac{1}{4u} + C\\ = \mathstrut \amp -\frac{1}{4\sin\theta} + C\\ = \mathstrut \amp -\frac{\csc\theta}{4} + C \end{align*}

We now use a reference right triangle as shown in Figure 5.7.10 to relate the angle \(\theta\) and the variable \(x\text{.}\) Doing this gives \(\csc\theta = \frac{\sqrt{4+x^2}}{x}\text{.}\) We conclude that

\begin{equation*} \int \frac{1}{x^2\sqrt{4+x^2}} \ dx = -\frac{\csc\theta}{4} + C = -\frac{\sqrt{4+x^2}}{4x} + C. \end{equation*}
