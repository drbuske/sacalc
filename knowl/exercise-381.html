Exercise 9.

State whether each statement is true or give an example to show that it is false.

  1. If \(\sum_{k=1}^{\infty} a_kx^k\) converges, then \(a_kx^k \rightarrow 0\) as \(k \rightarrow \infty\text{.}\)

  2. \(\sum_{k=1}^{\infty} a_kx^k\) converges at \(x=0\) for any real numbers \(a_k\text{.}\)

  3. Given any sequence \(a_k\text{,}\) there is always some \(R > 0\text{,}\) possibly very small, such that \(\sum_{k=1}^{\infty} a_kx^k\) converges on \((-R,R)\text{.}\)

  4. If \(\sum_{k=1}^{\infty} a_kx^k\) has radius of convergence \(R > 0\) and if \(\left| b_k \right| \leq \left| a_k \right|\) for all \(k\text{,}\) then the radius of convergence of \(\sum_{k=1}^{\infty} b_kx^k\) is greater than or equal to \(R\text{.}\)

  5. If \(\sum_{k=1}^{\infty} a_kx^k\) has radius of convergence \(R > 0\) and if \(\left| b_k \right| \leq \left| a_k \right|\) for all \(k\text{,}\) then the radius of convergence of \(\sum_{k=1}^{\infty} b_k(x-1)^k\) is greater than or equal to \(R\text{.}\)
