Here \(x=\cos\theta - \cos^2\theta\) and \(y=\sin\theta - \sin^2\theta\text{.}\) So

\begin{equation*} \frac{dy}{dx} = \frac{\cos\theta - \cos (2\theta)}{-\sin\theta + \sin 2\theta} =0 \end{equation*}

when \(\theta = 0, -\frac{2\pi}{3}, \frac{2\pi}{3}\text{.}\) So there are horizontal tangent lines at \((0,0)\text{,}\) \(\left(\frac{3}{2}, \frac{2\pi}{3}\right)\text{,}\) and \(\left(\frac{3}{2}, -\frac{2\pi}{3}\right)\text{.}\) \(\frac{dy}{dx} = 0\) exactly when \(\sin \theta = \sin (2\theta)\) or when \(\theta = -\pi, -\frac{\pi}{3}, 0, \frac{\pi}{3}, \pi\text{.}\) So there are vertical tangent lines at \((-2,0), \left(\frac{1}{2}, -\frac{\pi}{3}\right)\text{,}\) and \(\left(\frac{1}{2}, \frac{\pi}{3}\right)\text{.}\) Note that the tangent line at the origin is horizontal as \(\displaystyle \lim_{\theta \rightarrow 0} \frac{\cos\theta - \cos 2\theta}{-\sin\theta + \sin 2\theta} = 0\text{.}\)
