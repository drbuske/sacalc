Solution 7.4.4.5.1.

A. Noting that the sequence \(a_n = {1\over 4^n}\) is a sequence of positive decreasing terms, we know we can apply the Integral Test. We compare \(\sum\limits_{n=1}^{\infty} {1\over 4^n}\) with \(\int_c^{\infty} {1\over 4^n}\,dn\text{.}\) Taking \(c = 1\text{,}\) we have
\begin{equation*} \int_1^{\infty} {1\over 4^n}\,dn = \lim\limits_{b\to\infty} -{1\over \ln(4) 4^b} + {1\over 4 \ln(4)} = {1\over 4 \ln(4)}. \end{equation*}
Thus the integral, and the sum, converge. B. Again, we note that \(a_n = {n+7\over n^2 + 14 n + 9}\) is a sequence of positive decreasing terms, so that the Integral Test applies. We compare \(\sum\limits_{n=1}^{\infty} {n+7\over n^2 + 14 n + 9}\) with \(\int_c^{\infty} {n+7\over n^2 + 14 n + 9}\,dn\text{.}\) Taking \(c=1\text{,}\) we have
\begin{equation*} \int_1^{\infty} {n+7\over n^2 + 14 n + 9}\,dn = \lim\limits_{n\to\infty} {1\over 2}\ln(b^2 + 14 b + 9) - {1\over 2}\ln(24). \end{equation*}
Because \(\lim\limits_{n\to\infty} \ln(b^2 + 14 b + 9)\) diverges, we know that the integral, and thus the sum, diverge.
