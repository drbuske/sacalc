Item d.

The reason why this point did not show up as a solution is because the origin is on both graphs but for different values of \(\theta\text{.}\) For example, for the cardioid we get

\begin{align*} 2+2\sin\theta = \mathstrut \amp 0\\ \sin \theta = \mathstrut \amp -1 \end{align*}

so the values for \(\theta\) that solve this equation are \(\theta = \frac{3\pi}{2}+2n\pi\text{,}\) where \(n\) is any integer. For the circle we get

\begin{equation*} 6\sin\theta = 0. \end{equation*}

The solutions to this equation are of the form \(\theta = n\pi\) for any integer value of \(n\text{.}\) These two solution sets have no points in common. Regardless of this fact, the curves intersect at the origin.
