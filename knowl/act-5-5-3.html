Activity 5.5.3.

Evaluate each indefinite integral using partial fractions.

  1. \(\displaystyle \displaystyle \int \frac{x+1}{6x+x^2} \ dx\)

  2. \(\displaystyle \int \frac{1}{x^3-x^2} \ dx\text{;}\) use \(\displaystyle \frac{A}{x} + \frac{B}{x^2} + \frac{C}{x-1}\)

  3. \(\displaystyle \int \frac{10x+2}{x^3-5x^2+x-5} \ dx\text{;}\) use \(\displaystyle \frac{A}{x-5} + \frac{Bx+C}{x^2+1}\)

  4. \(\displaystyle \int \frac{x^4 + 12x^3 + 15x^2 + 25x+11}{x^3 + 12x^2+11x} \ dx\text{;}\) use division and \(\displaystyle \frac{A}{x} + \frac{B}{x+1} + \frac{C}{x+11}\)
