Solution 5.3.5.4.1.

\(w = e^{2 x} + 6\text{.}\) Then \(dw = 2\,e^{2 x}\,dx\text{,}\) so that
\begin{equation*} \int\,\frac{e^{2 x}}{6 + e^{2 x}}\,dx = {1\over 2}\int\,{1\over w}\,dw = {1\over 2}\,\ln(|w|) + C = {1\over 2}\,\ln(|e^{2 x} + 6|) + C. \end{equation*}
Because \(e^{2 x} + 6\) is always positive, we can drop the absolute values from the argument of the natural log.
