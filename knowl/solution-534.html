Solution 9.1.5.5.1.

\(g(t) = \ln\!\left(5t\right)\text{.}\) Then \(g'(t) = \frac{1}{t}\text{.}\) The particle is moving counterclockwise when \(g'(t)>0\text{,}\) that is, when \(t\) is in \(\left(0,\infty \right)\text{.}\) Any other values of \(t\text{,}\) \(t \le 0\text{,}\) are not in the domain of the function, so the particle is never moving clockwise. The full circle is traced out if \(g(t) = \ln\!\left(5t\right)\) produces values over one period of the functions \(sin(t)\) and \(cos(t)\text{.}\) In this case \(g(t)\) produces all values \(\left(-\infty ,\infty \right)\text{,}\) so the full circle must be produced. To find a point where the parameterization is tracing out the point \((1,0)\text{,}\) we are looking for a \(t\) such that \(g(t) = 0\text{.}\) Such a value is \(t = 0.2\text{.}\)
