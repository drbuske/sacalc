This integral cannot be evaluated using any of the techniques we have discussed so far. However, if we make the substitution \(x=3\sin\theta\text{,}\) we have \(dx = 3\cos\theta \ d\theta\text{.}\) After substituting into the integral, we have

\begin{align*} \int \sqrt{9-x^2} \ dx = \mathstrut \amp \int \sqrt{9-(3\sin\theta)^3}(3\cos\theta) \ d\theta \\ = \mathstrut \amp \int 9\sqrt{1-\sin^2\theta} \cos\theta \ d\theta \\ = \mathstrut \amp \int 9\sqrt{\cos^2\theta} \ d\theta. \end{align*}

Assuming that \(\cos\theta \geq 0\text{,}\) we have

\begin{equation*} \int \sqrt{9-x^2} \ dx = \int 9 \cos^2\theta \ d\theta. \end{equation*}

We can now evaluate the integral using the techniques developed for integrating powers and products of trigonometric functions. Since \(\sin\theta = \frac{x}{3}\text{,}\) we can construct the reference triangle shown in Figure 5.7.3.
