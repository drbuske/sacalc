Answer 9.2.5.1.

  1. \(\displaystyle \sqrt{[x(t_i)-x(t_{i-1})]^2 + [y(t_i)-y(t_{i-1})]^2}\)

  2. \(\displaystyle \displaystyle \sum_{i=1}^n \sqrt{[x(t_i)-x(t_{i-1})]^2 + [y(t_i)-y(t_{i-1})]^2}\)

  3. True

  4. We need to let \(n \rightarrow \infty\) by having \(\Delta t \rightarrow 0\text{.}\) The sum in part (c) then becomes

    \begin{align*} \int_0^{2\pi} \sqrt{\left( \frac{dx}{dt}\right)^2 + \left( \frac{dy}{dt} \right)^2} \ dt = \mathstrut \amp \int_0^{2\pi} \sqrt{(1-\cos t)^2 + (\sin t)^2} \ dt\\ = \mathstrut \amp \int_0^{2\pi} \sqrt{2-2\cos t} \ dt\\ = \mathstrut \amp 8. \end{align*}
