Solution 9.3.6.1.

Because the function is a multiple of a sine function, it is periodic with period \(2\pi\text{,}\) so use values for \(\theta\) between \(0\) and \(2\pi\) in creating a table (see Table 9.3.7) of ordered pairs to be plotted (see Figure 9.3.8).

This is the graph of a circle. The equation \(r=4\sin\theta\) can be converted into rectangular coordinates by first multiplying both sides by \(r\text{.}\) This gives the equation \(r^2=4r\sin\theta\text{.}\) Next use the facts that \(r^2=x^2+y^2\)and \(y=r\sin\theta\text{.}\) This gives \(x^2+y^2=4y\text{.}\) To put this equation into standard form, subtract \(4y\) from both sides of the equation and complete the square:

\begin{align*} x^2+y^2 - 4y = \mathstrut \amp 0\\ x^2 + (y^2-4y) = \mathstrut \amp 0\\ x^2+(y^2-4y+4) = \mathstrut \amp 0+4\\ x^2 + (y-2)^2 = \mathstrut \amp 4. \end{align*}

This is the equation of a circle with radius \(2\) and center \((0,2)\) in the rectangular coordinate system.
