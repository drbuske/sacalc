1. First, note that

    \begin{equation*} \frac{d}{dx} = \frac{1}{\sqrt{1-x^2}} = (1-x^2)^{-\frac{1}{2}} \end{equation*}

    for \(|x| < 1\text{.}\) Using the fact that

    \begin{equation*} (1+x)^p = 1+px + \frac{p(p-1)}{2!}x^2 + \frac{p(p-1)(p-2)}{3!}x^3 + \cdots\text{,} \end{equation*}

    we find that

    \begin{align*} (1-x^2)^{-\frac{1}{2}} \amp = 1 - \frac{1}{2}(-x^2) + \frac{ \left( -\frac{1}{2} \right) \left( -\frac{3}{2} \right)}{2!} (-x^2)^2 + \frac{ \left( -\frac{1}{2} \right) \left( -\frac{3}{2} \right) \left( -\frac{5}{2} \right) }{3!} (-x^2)^3 + \cdots \\ \amp = 1 + \frac{1}{2}x^2 + \frac{3}{8}x^4 + \frac{15}{48}x^6 + \cdots \end{align*}

    So, by integrating term by term and using the fact that \(\arcsin(0)=0\text{,}\) we see that

    \begin{equation*} \arcsin (x) = x + \frac{1}{6}x^3 + \frac{3}{40}x^5 + \frac{5}{112}x^7 + \cdots\text{.} \end{equation*}
  2. \begin{equation*} \sum_{p=1}^{\infty} px^{p-1} = \frac{d}{dx} \left( \sum_{p=0}^{\infty} x^p \right) = \frac{d}{dx} \left( \frac{1}{1-x} \right) = \frac{1}{(1-x)^2} \end{equation*}

    for \(|x| < 1\text{.}\)

  3. \begin{equation*} \frac{1}{2}e^x + \frac{1}{2}e^{-x} = \frac{1}{2}\sum_{k=0}^{\infty} \frac{x^k}{k!} + \frac{1}{2} \sum_{k=0}^{\infty} \frac{(-x)^k}{k!} = \sum_{k=0}^{\infty} \frac{x^{2k}}{(2k)!} \end{equation*}

    Since each term in this series is even, the function \(f(x)\) is even. The Taylor series for \(f'(x)\) is given by

    \begin{equation*} \sum_{k=1}^{\infty} \frac{x^{2k-1}}{(2k-1)!}\text{.} \end{equation*}

    Since each term is odd, the function \(f'(x)\) is odd. \(f''(x)\) will be even based on the fact that its Taylor series will only consist of even powers.
