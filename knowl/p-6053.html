By making the substitution \(x=a\sin\theta\text{,}\) we are thus able to convert an integral involving a radical into an integral involving trigonometric functions. After we evaluate this new integral, we can convert the solution back to an expression involving \(x\text{.}\) To see how this works, begin by assuming that \(0 < x < a\text{.}\) In this case, we have \(0 < \theta < \frac{\pi}{2}\text{.}\) Since \(\sin\theta = \frac{x}{a}\text{,}\) we can draw the reference triangle in Figure 5.7.6 to assist in expressing the values of all the trigonometric functions in terms of \(x\text{.}\) It can be shown that this triangle actually produces the correct values of the trigonometric functions evaluated at \(\theta\) for all \(\theta\) satisfying \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\text{.}\) Note that the expression \(\sqrt{a^2-x^2}\) appears as the length of one side of this reference triangle.
