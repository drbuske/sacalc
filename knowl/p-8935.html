ExampleÂ 8.1.5 gives us significant insight as to the process of determining the values of \(x\) for which a power series

\begin{equation*} \sum_{k =0}^{\infty} c_k(x-a)^k\text{,} \end{equation*}

centered at \(x=a\) will converge. In general we apply the Ratio Test with \(a_k = \left|c_k (x-a)^k \right|\text{.}\) This series converges if \(\lim_{k \rightarrow \infty} \frac{a_{k+1}}{a_k} < 1\text{.}\) But if we have \(\lim_{k \rightarrow \infty} \frac{\left| c_{k+1} \right|}{\left| c_k \right|} = L\text{,}\) then

\begin{align*} \lim_{k \rightarrow \infty} \frac{a_{k+1}}{a_k} \amp = \lim_{k \rightarrow \infty} \left| x-a \right| \frac{ \left| c_{k+1}\right|}{\left| c_k \right|}\\ \amp = \lim_{k \rightarrow \infty} \left| x-a \right| \cdot L \end{align*}

There are three possibilities for \(L\text{:}\) \(L\) can be \(0\text{,}\) it can be a finite positive value, or it can be infinite. Based on this value of \(L\text{,}\) we can determine for which values of \(x\) the original power series converges.

  • If \(L=0\text{,}\) then the power series converges on \((-\infty, \infty)\text{.}\)

  • If \(L\) is infinite, then the power series converges only at \(x=a\text{.}\)

  • If \(L\) is finite and nonzero, then the power series converges absolutely for all \(x\) that satisfy

    \begin{equation*} \left| x-a \right| \cdot L < 1 \end{equation*}

    or for all \(x\) such that

    \begin{equation*} \left| x-a \right| < \frac{1}{L}\text{,} \end{equation*}

    which is the interval

    \begin{equation*} \left( a - \frac{1}{L}, a+\frac{1}{L} \right)\text{.} \end{equation*}

    This interval is centered at \(x=a\) and has radius \(R=\frac{1}{L}\text{.}\) Because the Ratio Test is inconclusive when \(\left| x-a \right| \cdot L = 1\text{,}\) the endpoints \(x=a\pm \frac{1}{L}\) must be checked separately.
