Item b.

Define \(a_k = \frac{k-1}{k^2+3}\text{.}\) Choose \(b_k = \frac{k}{k^3} = \frac{1}{k^2}\) so that both series are positive for \(k \geq 2\text{.}\) Then,

\begin{equation*} a_k = \frac{k-1}{k^3+3} \leq \frac{k}{k^3} = b_k \end{equation*}

so that \(a_k \leq b_k\) for \(k \geq 2\text{.}\) Since \(\sum b_k\) converges, so does \(\sum a_k\text{.}\)
