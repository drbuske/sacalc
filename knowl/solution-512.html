Solution 8.3.5.8.1.

\(\cos(1)\) using the \(n^{\rm th}\) degree polynomial, the error is at most \(\frac{1}{(n+1)!}\text{.}\) For the answer to be correct to four decimal places, the error must be less than \(5\times 10^{-6}\text{.}\) Thus, the first \(n\) such that \(\frac{1}{(n+1)!} \lt 5\times 10^{-6}\) will work. In particular, this is first true when \(n=8\text{.}\) For 8 decimal places, we need \(\frac{1}{(n+1)!} \lt 5\times 10^{-9}\text{,}\) for which \(n=11\) works.
