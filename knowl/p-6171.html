  1. WolframAlpha gives \(\sinh^{-1}(x)\) or \(\ln(\sqrt{x^2+1}+x)+C\text{.}\) GeoGebra returns \(-\ln(-x+\sqrt{x^2+1})+C\text{.}\) These are found to be equal by rationalizing \((-x+\sqrt{x^2+1})^{-1}\text{.}\)

  2. WolframAlpha gives \(\frac{1}{12}\cos(3x)-\frac{3}{4}\cos(x)+C\text{.}\) GeoGebra returns \(-\cos(x)+\frac{1}{3}\cos^3(x)+C\text{.}\) These are found to be equal by through trigonometric identities (or visually by simply plotting the difference).

  3. WolframAlpha gives \(2\sqrt{x}-3\sqrt[3]{x} + 6\sqrt[6]{x} - 6\ln(\sqrt[6]{x}+1)+C\text{.}\) GeoGebra returns \(6[-\frac{1}{2}\sqrt[3]{x}-\ln(\sqrt[6]{x}+1)+\frac{1}{3}\sqrt{x}+\sqrt[6]{x}]+C\text{.}\) These are found to be equal by simple algebraic simplification.

  4. WolframAlpha gives \(\frac{2}{\cos\left( \frac{x}{2}\right)+1}+C\text{.}\) GeoGebra returns \(\frac{-2}{\tan \left(\frac{1}{2}x\right)+1}+C\text{.}\) These are not actually equal; however, they do differ by a constant. That is, \(\frac{2}{\cot\left(\frac{x}{2}\right)+1} - 2 = \frac{-2}{\tan\left( \frac{x}{2}\right) + 1}\text{.}\)
