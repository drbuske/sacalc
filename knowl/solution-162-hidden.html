\((3,4)\) is on the curve, we know \(f(3)=4\text{.}\) The point \((3.2,4.3)\) is on the tangent line, so
\begin{equation*} \hbox{Slope of tangent} = \frac{4.3-4}{3.2-3} = \frac{0.3}{0.2} = 1.5. \end{equation*}
Thus, \(f'(3)=1.5\text{.}\) By the chain rule
\begin{equation*} h'(3) = 5(f(3))^{4}\cdot f'(3) =5\cdot 4^{4}\cdot 1.5 = 1920. \end{equation*}
Because \(g(x)\) is the inverse function of \(f\text{,}\) we know that \(g(4) = 3\text{.}\) Then note that \(f(g(x)) = f(f^{-1}(x)) = x\text{,}\) so \(f(g(x)) = x\text{.}\) Differentiating both sides of the equation, \(f'(g(x))\cdot g'(x) = 1\text{,}\) so \(g'(x) = \frac{1}{f'(g(x))}\text{.}\) Thus \(g'(4) = \frac{1}{f'(g(4))} = \frac{1}{f'(3)} = \frac1{1.5}\text{.}\)
