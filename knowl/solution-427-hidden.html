\(\int_0^{\pi/2}\,{\cos(t)\over\sin(t)}\,dt\text{.}\) However, the integrand is undefined at \(t = 0\text{,}\) so this is an improper integral. We find
\begin{equation*} \int_0^{\pi/2}\,{\cos(t)\over\sin(t)}\,dt = \lim\limits_{a\to0^{+}}\,\int_a^{\pi/2}\,{\cos(t)\over\sin(t)}\,dt = \end{equation*}
\begin{equation*} \lim\limits_{a\to0^{+}}\,(\ln|\sin(t)|)\bigg|_a^{\pi/2} = \lim\limits_{a\to0^{+}} ( \ln(1) - \ln|\sin(a)| ). \end{equation*}
However, \(-\ln|\sin(a)|\) is undefined as \(a\to0\text{,}\) so this integral diverges. Thus, because the integral diverges, the area is not bounded.
