Section 5.7 Trigonometric Substitution

Motivating Questions

  • How does the method of trigonometric substitution help us find antiderivatives of functions that may include expressions including \(\sqrt{a^2-x^2}\text{,}\) \(a^2+x^2\text{,}\) and \(\sqrt{a^2+x^2}\) where \(a\) is any real number?

We've now studied three integration techniques: \(u\)-substitution, integration by parts, and the method of partial fractions. However, none of these may be used in situations involving rational functions with a single quadratic factor in the denominator like \(\int \frac{1}{4+x^2} \ dx\) or when integrating many functions containing expressions like \(\sqrt{9-x^2}\text{.}\)

  4. Write the expression \(\displaystyle \frac{x^3}{a^2\sqrt{a^2-x^2}}\) in terms of \(\theta\text{.}\)

Subsection 5.7.1 Integrals Involving \(\sqrt{a^2-x^2}\)

Before developing a general strategy for integrals containing \(\sqrt{a^2-x^2}\text{,}\) we look at an example.

Example 5.7.2.

Evaluate

\begin{equation*} \displaystyle \int \sqrt{9-x^2} \ dx. \end{equation*}
Solution.

This integral cannot be evaluated using any of the techniques we have discussed so far. However, if we make the substitution \(x=3\sin\theta\text{,}\) we have \(dx = 3\cos\theta \ d\theta\text{.}\) After substituting into the integral, we have

\begin{align*} \int \sqrt{9-x^2} \ dx = \mathstrut \amp \int \sqrt{9-(3\sin\theta)^3}(3\cos\theta) \ d\theta \\ = \mathstrut \amp \int 9\sqrt{1-\sin^2\theta} \cos\theta \ d\theta \\ = \mathstrut \amp \int 9\sqrt{\cos^2\theta} \ d\theta. \end{align*}

Assuming that \(\cos\theta \geq 0\text{,}\) we have

\begin{equation*} \int \sqrt{9-x^2} \ dx = \int 9 \cos^2\theta \ d\theta. \end{equation*}

We can now evaluate the integral using the techniques developed for integrating powers and products of trigonometric functions. Since \(\sin\theta = \frac{x}{3}\text{,}\) we can construct the reference triangle shown in Figure 5.7.3.

Thus

\begin{align*} \int \sqrt{9-x^2} \ dx = \mathstrut \amp \int 9\sqrt{\cos^2\theta} \ d\theta \\ = \mathstrut \amp 9\left(\frac{1}{2}+\frac{1}{2}\cos(2\theta)\right) \ d\theta \\ = \mathstrut \amp \frac{9}{2}\theta + \frac{9}{4}\sin(2\theta) + C\\ = \mathstrut \amp \frac{9}{2}\theta + \frac{9}{4}(2\sin\theta\cos\theta) + C\\ = \mathstrut \amp \frac{9}{2}\arcsin\left(\frac{x}{3}\right) + \frac{9}{2}\cdot \frac{x}{3} \cdot \frac{\sqrt{9-x^2}}{3} + C\\ = \mathstrut \amp \frac{9}{2}\arcsin\left(\frac{x}{3}\right) + \frac{x\sqrt{9-x^2}}{2} + C \end{align*}

Example 5.7.4.

Evaluate

\begin{equation*} \displaystyle \int \frac{\sqrt{4-x^2}}{x} \ dx. \end{equation*}
Solution.

First make the substitution \(x=2\sin\theta\text{.}\) Then \(dx = 2\cos\theta \ d\theta\text{.}\) Since \(\sin \theta = \frac{x}{2}\text{,}\) we can construct the reference triangle shown in Figure 5.7.5.

Thus,

\begin{align*} \int \frac{\sqrt{4-x^2}}{x} \ dx = \mathstrut \amp \int \frac{\sqrt{4-(2\sin\theta)^2}}{2\sin\theta} \ \cdot 2\cos\theta \ d\theta\\ = \mathstrut \amp \int \frac{2\cos^2\theta}{\sin\theta} \ d\theta\\ = \mathstrut \amp \int \frac{2(1-\sin^2\theta)}{\sin\theta} \ d\theta\\ = \mathstrut \amp \int 2\csc\theta - 2\sin\theta \ d\theta\\ = \mathstrut \amp 2\ln|\csc\theta - \cot\theta| + 2\cos\theta + C\\ = \mathstrut \amp 2\ln\left\lvert\frac{2}{x} - \frac{\sqrt{4-x^2}}{x}\right\rvert + \sqrt{4-x^2} + C. \end{align*}

The general trigonometric substitution process for evaluating integrals involving \(\sqrt{a^2-x^2}\) makes use of the substitution \(x=a\sin\theta\) so that \(dx = a\cos\theta\text{.}\) The domain of \(\sqrt{a^2-x^2}\) is \([-a,a]\) so that \(-a \leq x \leq a\) or \(-1 \leq \frac{x}{a} \leq 1\text{.}\) Since the range of \(\sin x\) over \([-\pi/2,\pi/2]\) is \([-1,1]\text{,}\) there is a unique angle \(\theta\) satisfying \(-\pi/2 \leq \theta \leq \pi/2\) so that \(\sin\theta = \frac{x}{a}\text{,}\) or equivalently, so that \(x=a\sin\theta\text{.}\) If we substitute \(x=a\sin\theta\) into \(\sqrt{a^2-x^2}\text{,}\) we get

\begin{align*} \sqrt{a^2-x^2} = \mathstrut \amp \sqrt{a^2-(a\sin\theta)^2}\\ = \mathstrut \amp \sqrt{a^2-a^2\sin^2\theta}\\ = \mathstrut \amp \sqrt{a^2(1-\sin^2\theta)}\\ = \mathstrut \amp \sqrt{a^2\cos^2\theta}\\ = \mathstrut \amp |a\cos\theta|\\ = \mathstrut \amp a\cos\theta \end{align*}

where \(|a\cos\theta| = a\cos\theta\) since \(\cos\theta \geq 0\) on \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\text{.}\)

By making the substitution \(x=a\sin\theta\text{,}\) we are thus able to convert an integral involving a radical into an integral involving trigonometric functions. After we evaluate this new integral, we can convert the solution back to an expression involving \(x\text{.}\) To see how this works, begin by assuming that \(0 < x < a\text{.}\) In this case, we have \(0 < \theta < \frac{\pi}{2}\text{.}\) Since \(\sin\theta = \frac{x}{a}\text{,}\) we can draw the reference triangle in Figure 5.7.6 to assist in expressing the values of all the trigonometric functions in terms of \(x\text{.}\) It can be shown that this triangle actually produces the correct values of the trigonometric functions evaluated at \(\theta\) for all \(\theta\) satisfying \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\text{.}\) Note that the expression \(\sqrt{a^2-x^2}\) appears as the length of one side of this reference triangle.

Integrating Expressions Involving \(\sqrt{a^2-x^2}\).

  • Make the substitution \(x=a\sin\theta\) so that \(dx=a\cos\theta \ d\theta\text{.}\) It follows that \(\sqrt{a^2-x^2} = a\cos\theta\text{.}\)

  • Simplify the integrand which is now in the variable \(\theta\text{.}\)

  • Evaluate the integral using techniques from Section 5.6.

  • Use the reference triangle from Figure 5.7.6 to rewrite the result in terms of \(x\text{.}\)

Example 5.7.7.

Evaluate

\begin{equation*} \displaystyle \int \frac{1}{\sqrt{9-4x^2}} \ dx. \end{equation*}
Solution.

Since \(\sqrt{9-4x^2} = \sqrt{9-(2x)^2}\text{,}\) we will want to substitute

\begin{equation} 2x=3\sin\theta\tag{5.21} \end{equation}

so that we can factor out the 9 and obtain the expression \(1-\sin^2\theta\text{.}\) Assuming \(2x=3\sin\theta\text{,}\) then \((2x)^2=9\sin^2\theta\) and \(2 \ dx = 3\cos\theta \ d\theta\text{.}\) Then,

\begin{align*} \int \frac{1}{\sqrt{9-4x^2}} \ dx = \mathstrut \amp \int \frac{1}{\sqrt{9-(2x)^2}} \ dx\\ = \mathstrut \amp \int \frac{(3/2)\cos\theta}{\sqrt{9-9\sin^2\theta}} \ d\theta\\ = \mathstrut \amp \frac{3}{2}\int \frac{\cos\theta}{\sqrt{9(1-\sin^2\theta)}} \ d\theta\\ = \mathstrut \amp \frac{3}{2}\int \frac{\cos\theta}{3\sqrt{\cos^2\theta}} \ d\theta\\ = \mathstrut \amp \frac{3}{2}\int \frac{\cos\theta}{3\cos\theta} \ d\theta\\ = \mathstrut \amp \frac{3}{6}\int 1 \ d\theta\\ = \mathstrut \amp \frac{1}{2}\theta + C. \end{align*}

We can solve for \(\theta\) in Equation (5.21) to find

\begin{equation*} \int \frac{1}{\sqrt{9-4x^2}} \ dx = \frac{1}{2}\arcsin \left( \frac{2x}{3} \right) + C. \end{equation*}

Subsection 5.7.2 Integrals Involving \(a^2+x^2\)

Just as the substitution \(x=a\sin\theta\) can be useful when \(a^2-x^2\) appears in the integrand, the trigonometric substitution \(x=a\tan\theta\) may be helpful when dealing with terms involving \(a^2+x^2\text{.}\)

Example 5.7.8.

Evaluate

\begin{equation*} \displaystyle \int \frac{1}{1+x^2} \ dx. \end{equation*}
Solution.

The trigonometric substitution

\begin{equation} x=\tan\theta\tag{5.22} \end{equation}

gives \(dx = \sec^2\theta \ d\theta\text{.}\) Then,

\begin{align*} \int \frac{1}{1+x^2} \ dx = \mathstrut \amp \int \frac{\sec^2\theta}{1+\tan^2\theta} \ d\theta\\ = \mathstrut \amp \int \frac{\sec^2\theta}{\sec^2\theta} \ d\theta\\ = \mathstrut \amp \int 1 \ d\theta\\ = \mathstrut \amp \theta + C. \end{align*}

We can solve for \(\theta\) in Equation (5.22) to find \(\theta = \arctan(x)\) and hence

\begin{equation*} \int \frac{1}{1+x^2} \ dx = \arctan(x) + C. \end{equation*}

Example 5.7.9.

Evaluate

\begin{equation*} \displaystyle \int \frac{1}{x^2\sqrt{4+x^2}} \ dx. \end{equation*}
Solution.

Set

\begin{equation} x=2\tan\theta\tag{5.23} \end{equation}

so that \(dx = 2\sec^2\theta \ d\theta\text{.}\) Since \(x^2=4\tan^2\theta\text{,}\) we use the identity \(1+\tan^2\theta = \sec^2\theta\) to get

\begin{align*} \int \frac{1}{x^2\sqrt{4+x^2}} \ dx = \mathstrut \amp \int \frac{2\sec^2\theta}{4\tan^2\theta \sqrt{4+4\tan^2\theta}} \ d\theta\\ = \mathstrut \amp \int \frac{2sec^2\theta}{4\tan^2\theta \cdot 2\sec\theta} \ d\theta\\ = \mathstrut \amp \frac{1}{4} \int \frac{\sec\theta}{\tan^2\theta} \ d\theta\\ = \mathstrut \amp \frac{1}{4} \int \frac{1}{\cos\theta} \frac{\cos^2\theta}{\sin^2\theta} \ d\theta\\ = \mathstrut \amp \frac{1}{4} \int \frac{\cos\theta}{\sin^2\theta} \ d\theta. \end{align*}

This last integral can be evaluated via yet another substitution. This time, a \(u\)-substitution in which we set \(u=\sin\theta\) so that \(du = \cos\theta \ d\theta\) will be our path forward. Doing so gives

\begin{align*} \int \frac{1}{x^2\sqrt{4+x^2}} \ dx = \mathstrut \amp \frac{1}{4} \int \frac{\cos\theta}{\sin^2\theta} \ d\theta\\ = \mathstrut \amp \frac{1}{4} \int \frac{1}{u^2} \ du\\ = \mathstrut \amp -\frac{1}{4u} + C\\ = \mathstrut \amp -\frac{1}{4\sin\theta} + C\\ = \mathstrut \amp -\frac{\csc\theta}{4} + C \end{align*}

We now use a reference right triangle as shown in Figure 5.7.10 to relate the angle \(\theta\) and the variable \(x\text{.}\) Doing this gives \(\csc\theta = \frac{\sqrt{4+x^2}}{x}\text{.}\) We conclude that

\begin{equation*} \int \frac{1}{x^2\sqrt{4+x^2}} \ dx = -\frac{\csc\theta}{4} + C = -\frac{\sqrt{4+x^2}}{4x} + C. \end{equation*}

Subsection 5.7.3 Combined Methods

Trigonometric substitution can be combined with partial fractions, completing the square, \(u\)-substitution, and more to address an ever wider set of integrals.

Example 5.7.11.

Evaluate

\begin{equation*} \displaystyle \int \frac{2x+7}{x^2+6x+18} \ dx. \end{equation*}
Solution.

We start by completing the square in the denominator to find

\begin{align*} \int \frac{2x+7}{x^2+6x+18} \ dx = \mathstrut \amp \int \frac{2x+7}{(x+3)^2+9} \ dx\\ = \mathstrut \amp \int \frac{2(u-3)+7}{u^2+9} \ du\\ = \mathstrut \amp \int \frac{2u}{u^2+9} \ du + \int \frac{4}{u^2+9} \ du \end{align*}

where the substitution \(u=x+3\) has been employed. For the first of these integrals, use the substitution \(w=u^2+9\) and \(dw = 2u \ du\text{.}\) Then,

\begin{align*} \int \frac{2u}{u^2+9} \ du = \mathstrut \amp \int \frac{1}{w} \ dw\\ = \mathstrut \amp \ln |w| + C_1\\ = \mathstrut \amp \ln |(x+3)^2 + 9| + C_1 \end{align*}

For the second integral, \(\int \frac{4}{u^2+9} \ du\text{,}\) we can use a trigonometric substitution or we can factor out the 9, use the substitution \(v=w/3\) and \(dv = \frac{1}{3} \ dw\) and use the antidifferentiation rule \(\int \frac{1}{1+u^2} \ du = \arctan(u)+C\text{.}\) This latter technique gives

\begin{align*} \int \frac{4}{u^2+9} \ du = \mathstrut \amp \frac{4}{9} \int \frac{1}{(u/3)^2+1} \ du\\ = \mathstrut \amp \frac{4}{3} \int \frac{1}{v^2+1} \ dv\\ = \mathstrut \amp \frac{4}{3} \arctan(v) + C_2\\ = \mathstrut \amp \frac{4}{3} \arctan\left( \frac{x+3}{3}\right) + C_2 \end{align*}

Therefore,

\begin{equation*} \int \frac{2x+7}{x^2+6x+18} \ dx = \ln |(x+3)^2+9| + \frac{4}{3}\arctan \left( \frac{x+3}{3} \right) + C\text{.} \end{equation*}

Subsection 5.7.4 Summary

Trigonometric substitutions of the form \(u=\sin x\text{,}\) \(u=\tan x\text{,}\) or variants thereof let us find integrals like \(\int \frac{1}{\sqrt{1-x^2}} \ dx\) and \(\int \frac{1}{1+x^2} \ dx\text{.}\) A tangent-based substitution is essential for finding integrals of the form \(\frac{Ax+B}{ax^2+bx+c}\) where \(ax^2+bx+c\) is an irreducible quadratic.
